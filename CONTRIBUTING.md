# Contributing to Awesome Data Definitions

Thank you for your interest! This repository contains **sample definitions** from [mdatool.com](https://www.mdatool.com).

## How to Contribute

### 1. Suggest New Definitions
- Open an issue using the "Suggest a Definition" template
- We review all suggestions for inclusion in the full database

### 2. Report Errors
- Found an incorrect definition? Use the "Report an Error" template
- Include source/reference for the correction

### 3. Share Use Cases
- Tell us how you're using these definitions!
- Open a discussion or issue

## For Enterprise Contributions

If you represent an organization and want to contribute industry-specific definitions:
- Email: admin@mdatool.com
- We offer enterprise partnerships

## Code of Conduct

Be respectful, constructive, and professional. We're all here to improve data standards together.

**Note:** This is a sample repository. The full database with 17,000+ definitions is at [mdatool.com](https://www.mdatool.com).
